# Logo Assets

Use these files for GitHub README, social previews, and docs.

## Files

- `entra-ca-manager-wordmark.svg`: primary light-background banner/logo
- `entra-ca-manager-wordmark-dark.svg`: dark-background banner/logo
- `entra-ca-manager-logo.svg`: primary square icon
- `entra-ca-manager-logo-monochrome.svg`: single-color square icon

## Recommended Sizes

- GitHub README header: wordmark at `680-820px` width
- Repo/profile social card mockups: wordmark at `900-1100px` width
- Favicon/app icon source export: square icon at `512x512` (then downscale)
- Small docs/sidebar icon: square icon at `20-32px`
